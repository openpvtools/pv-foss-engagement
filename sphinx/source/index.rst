Engagement Statistics for FOSS PV Software
==========================================

Tracking engagement for open-source software is challenging, but
engagement statistics are helpful in securing development and
maintenance funding.

This website summarizes datasets like GitHub star count and
ReadTheDocs page view analytics for many open-source
software packages in the solar photovoltaics space.
